# Documentation

## Milestones

### 1. Cpu architecture

- [ ] **Memory buffer:** Initialize `memory: [u8; 4096]`.
- [ ] **CPU boot state:** Initialize 16-bit Program Counter (`PC`) to `0x200`.
- [ ] Index: Initialize 16-bit Index Register (`I`).
- [ ] Refresh rate: Initialize 8-bit Delay Timer (`60Hz`).
- [ ] Sound: Initialize 8-bit Sound Timer.
- [ ] Registers: Initialize 16 8-bit general-purpose variable registers (`V0...VF`).

## Specifications

### Memory map

- `0x000..=0x04F` — Built-in font sprites (80 bytes)
- `0x200..=0x0FFF` — Program / data area

### Built-in font set (80 bytes)

Stored at: `FONT_BASE = 0x000` (range `0x000..=0x04F`)
Layout: 16 glyphs (`0x0..=0xF`), each glyph is 5 bytes.

```rust
// 0-F font sprites, 80 bytes total
pub const FONT_BASE: usize = 0x000;

pub const FONTSET: [u8; 80] = [
    0xF0, 0x90, 0x90, 0x90, 0xF0, // 0
    0x20, 0x60, 0x20, 0x20, 0x70, // 1
    0xF0, 0x10, 0xF0, 0x80, 0xF0, // 2
    0xF0, 0x10, 0xF0, 0x10, 0xF0, // 3
    0x90, 0x90, 0xF0, 0x10, 0x10, // 4
    0xF0, 0x80, 0xF0, 0x10, 0xF0, // 5
    0xF0, 0x80, 0xF0, 0x90, 0xF0, // 6
    0xF0, 0x10, 0x20, 0x40, 0x40, // 7
    0xF0, 0x90, 0xF0, 0x90, 0xF0, // 8
    0xF0, 0x90, 0xF0, 0x10, 0xF0, // 9
    0xF0, 0x90, 0xF0, 0x90, 0x90, // A
    0xE0, 0x90, 0xE0, 0x90, 0xE0, // B
    0xF0, 0x80, 0x80, 0x80, 0xF0, // C
    0xE0, 0x90, 0x90, 0x90, 0xE0, // D
    0xF0, 0x80, 0xF0, 0x80, 0xF0, // E
    0xF0, 0x80, 0xF0, 0x80, 0x80, // F
];
```

## Constraints

- RAM is fixed-size: 4096 bytes (`[u8; 4096]`).
- Programs start at `0x200` (initial PC).
